# Bedrock Skin Restorer
https://modrinth.com/plugin/bedrock-skin-restorer

![image](https://www.image2url.com/r2/default/images/1780799305993-b1ad184b-8992-4ee7-acfd-3253f9c7060b.png)

The GitHub repository for the plugin. Submit bugs here.

The plugin simply show a Bedrock player's skin on your server, as Floodgate is currently unreliable in doing its job.
It grabs your raw skin file via GeyserMC's Global API and overrides the player's default skin with the corrected version upon certain actions modifiable via a config.
